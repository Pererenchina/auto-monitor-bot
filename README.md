# Авто-Монитор Telegram Bot

Telegram-бот для мониторинга объявлений о продаже автомобилей на белорусских сайтах.

## Поддерживаемые сайты
- av.by
- kufar.by (раздел Авто)
- cars.onliner.by
- abw.by

## Возможности
- Настройка фильтров через inline-кнопки
- Автоматический мониторинг каждые 2-5 минут
- Уведомления о новых объявлениях
- Защита от дублей

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example` и укажите токен бота:
```
BOT_TOKEN=your_token_here
```

3. Запустите бота:
```bash
python main.py
```

## Использование

1. Запустите бота командой `/start`
2. Настройте фильтры через меню
3. Получайте уведомления о новых объявлениях
